🔒 Network Setup Required

Install Security Certificate to Access the Internet

Why do I need this?

To access the internet through this network, you must install a security certificate. This is required for:

  • ✓ Network security and content filtering
  • ✓ Malware and threat protection
  • ✓ Compliance with security policies

Step 1: Download Certificate for Your Device

🪟 Windows Installation Instructions

  1. The certificate file ca.crt should now be downloaded
  2. Find the downloaded file (usually in your Downloads folder)
  3. Double-click the ca.crt file
  4. Click "Install Certificate"
  5. Select "Local Machine" (you'll need administrator rights)
  6. Click "Yes" on the security prompt
  7. Choose "Place all certificates in the following store"
  8. Click "Browse" and select "Trusted Root Certification Authorities"
  9. Click "Next" → "Finish"
  10. You should see "The import was successful"

🤖 Android Installation Instructions

  1. The certificate file should now be downloaded
  2. Open "Settings"
  3. Go to "Security" (or "Biometrics and Security")
  4. Tap "Install from storage" or "Install certificates"
  5. Tap "CA certificate"
  6. You may see a warning - tap "Install anyway"
  7. Find and select the downloaded ca.crt file
  8. Give it a name (e.g., "SafeOps Network")
  9. Tap "OK"

🍎 iOS/iPadOS Installation Instructions

  1. The certificate profile should now be downloaded
  2. Open "Settings" app
  3. A banner should appear at the top saying "Profile Downloaded"
  4. Tap on the banner (or go to General → VPN & Device Management)
  5. Tap the downloaded profile
  6. Tap "Install" (top right)
  7. Enter your passcode if prompted
  8. Tap "Install" again (you'll see a red warning - this is normal)
  9. Tap "Done"
  10. IMPORTANT: Go to Settings → General → About → Certificate Trust Settings
  11. Toggle ON the certificate under "Enable Full Trust"
  12. Tap "Continue"
⚠️ Don't skip step 10-11! iOS requires this extra step to trust the certificate.

🖥️ macOS Installation Instructions

  1. The certificate file ca.crt should now be downloaded
  2. Find the file in your Downloads folder
  3. Double-click the ca.crt file
  4. Keychain Access will open
  5. Make sure "login" or "System" is selected
  6. Click "Add"
  7. Find the certificate in Keychain Access (look for "SafeOps")
  8. Double-click the certificate
  9. Expand "Trust"
  10. Set "When using this certificate" to "Always Trust"
  11. Close the window and enter your password

🐧 Linux Installation Instructions

Ubuntu/Debian:

  1. Download the certificate
  2. Open Terminal
  3. Run: sudo cp ~/Downloads/ca.crt /usr/local/share/ca-certificates/safeops-ca.crt
  4. Run: sudo update-ca-certificates
  5. Restart your browser

Fedora/RHEL/CentOS:

  1. Download the certificate
  2. Open Terminal
  3. Run: sudo cp ~/Downloads/ca.crt /etc/pki/ca-trust/source/anchors/safeops-ca.crt
  4. Run: sudo update-ca-trust
  5. Restart your browser
