Install Security Certificate

To access the internet through this network, you need to install our security certificate. This allows us to protect you from malicious websites.

Browser Warning Expected
You'll see a security warning. Click "Advanced" → "Proceed anyway" to continue.
Download Certificate for iOS

Installation Steps

  1. Tap Download Certificate above
  2. Go to Settings → General → VPN & Device Management
  3. Tap on SafeOps Root CA under "Downloaded Profile"
  4. Tap Install and enter your passcode
  5. Go to Settings → General → About → Certificate Trust Settings
  6. Enable full trust for SafeOps Root CA
  7. Return here - we'll automatically detect when you're done!
Download Certificate for Android

Installation Steps

  1. Tap Download Certificate above
  2. Open the downloaded file (or go to notification)
  3. When prompted, enter a name like "SafeOps"
  4. Select VPN and apps for certificate use
  5. Tap OK to install
  6. You may need to set a screen lock if not already set
  7. Return here - we'll automatically detect when you're done!
Download Certificate for Windows

Installation Steps

  1. Click Download Certificate above
  2. Double-click the downloaded file
  3. Click Install Certificate
  4. Select Local Machine and click Next
  5. Select Place all certificates in the following store
  6. Click Browse and choose Trusted Root Certification Authorities
  7. Click Next, then Finish
  8. Click Yes when prompted about trusting the certificate
  9. Return here - we'll automatically detect when you're done!
Download Certificate for macOS

Installation Steps

  1. Click Download Certificate above
  2. Double-click the downloaded file to open Keychain Access
  3. Enter your password when prompted
  4. Find SafeOps Root CA in the login keychain
  5. Double-click the certificate
  6. Expand Trust section
  7. Set "When using this certificate" to Always Trust
  8. Close and enter password to confirm
  9. Return here - we'll automatically detect when you're done!
Download Certificate for Linux

Installation Steps (Ubuntu/Debian)

  1. Click Download Certificate above
  2. Open Terminal and run:
  3. sudo cp ~/Downloads/SafeOps_Root_CA.crt /usr/local/share/ca-certificates/
  4. sudo update-ca-certificates
  5. For Firefox, import separately via Preferences → Privacy & Security → Certificates
  6. Return here - we'll automatically detect when you're done!
