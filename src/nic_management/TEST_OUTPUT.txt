================================================================================
                     WHAT HAPPENS WHEN YOU RUN NIC SERVICE
================================================================================

STARTUP SEQUENCE:
1. Load config.yaml (gRPC port, WAN/LAN interfaces, NAT settings)
2. Initialize Discovery Monitor → Auto-detect NICs
3. Create NAT Engine → Port allocator, mapping table
4. Start Connection Tracker → TCP/UDP/ICMP state machine
5. Initialize Load Balancer → WAN selection algorithms
6. Start Failover Handler → Health monitoring
7. Register Integration Hooks → Firewall, IDS, QoS
8. Start gRPC Server on port 50054
9. Begin packet processing loop

RUNTIME OPERATION:
- Outbound: LAN packet → Route lookup → NAT translation → Forward to WAN
- Inbound: WAN packet → NAT reverse lookup → Forward to LAN
- Failover: WAN health check fails 3x → Switch to backup WAN
- Recovery: Primary WAN recovers → Wait 60s → Switch back
