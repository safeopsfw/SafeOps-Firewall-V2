 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: Service won't start
Action:  Check [service]-error.log for details

Problem: No device detected after 60 seconds
Action:  Verify device is connected to 192.168.137.x network

Problem: Stuck at Stage 7 (DNS redirect)
Action:  Tell user to open browser manually

Problem: Stuck at Stage 8 (Portal access)
Action:  Tell user they should see install page in browser

Problem: Stuck at Stage 9 (Download)
Action:  Tell user to click "Download Certificate" button
